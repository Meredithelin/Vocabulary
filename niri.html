一、为什么选 Niri

Niri 是一个用 Rust 编写的滚动式平铺 Wayland 合成器,窗口排在一条无限延伸的水平条带上,新窗口不会挤压旧窗口,工作区按显示器独立纵向排列。相比 Hyprland 的"花哨特效路线"和 Sway 的"i3 复刻路线",Niri 的卖点是稳定、低输入延迟、滚动交互,以及对多显示器的优雅处理。

需要心里有数: Niri 本身不内置窗口模糊/阴影/动画粒子。"透明模糊"主要靠终端自身(Kitty/Ghostty)+ Waybar/Fuzzel/Mako 的 CSS 半透明 + 壁纸虚化营造统一玻璃感。如果非要 Hyprland 级特效,可以搭配 DankMaterialShell 或 Noctalia 外壳。

二、基础安装

假定你已经装好 Arch Linux 基础系统(使用 archinstall 或手动均可),用户已加入 wheel 组并能 sudo。

1. 启用 multilib 与 chaotic-aur(可选)

sudo pacman -Syu
sudo pacman -S --needed base-devel git

2. 安装 yay(AUR 助手)

cd /tmp
git clone https://aur.archlinux.org/yay-bin.git
cd yay-bin && makepkg -si

3. 安装 Niri 与全套伴生软件

sudo pacman -S niri kitty waybar fuzzel mako swaybg \
    grim slurp wl-clipboard swappy brightnessctl \
    pipewire pipewire-pulse wireplumber \
    xdg-desktop-portal xdg-desktop-portal-gnome xdg-desktop-portal-gtk \
    polkit-gnome gnome-keyring qt5-wayland qt6-wayland \
    noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-jetbrains-mono-nerd \
    papirus-icon-theme thunar mpv playerctl pavucontrol

# AUR 部分
yay -S swww hyprlock hypridle hyprpicker xwayland-satellite \
    bibata-cursor-theme catppuccin-gtk-theme-mocha
提示: xwayland-satellite 让 Niri 能跑 X11 程序(微信、QQ、JetBrains 全家桶等);xdg-desktop-portal-gnome 提供截屏/录屏功能。

4. 启动 Niri

从 TTY 登录后输入 niri-session;或使用 GDM/SDDM 选择 "Niri" 会话项。

八、壁纸 swww(带过渡动画)

# 启动守护进程(已在 niri config 里 spawn)
swww-daemon &

# 设置壁纸,带渐隐动画
swww img ~/Pictures/wallpaper.jpg \
    --transition-type wipe \
    --transition-duration 1.5 \
    --transition-fps 60

# 随机切换脚本(放进 ~/.local/bin/wallrand)
#!/usr/bin/env bash
DIR="$HOME/Pictures/Wallpapers"
swww img "$(find "$DIR" -type f | shuf -n 1)" \
    --transition-type grow --transition-pos 0.85,0.85 \
    --transition-duration 2

推荐去 wallhaven.cc 找 Catppuccin/暗紫色调的高分辨率壁纸,与上面的配色融合最和谐。

十二、进阶:DankMaterialShell 一键玻璃化

如果你嫌一个个组件配置麻烦,可以用 DankMaterialShell——它是基于 Quickshell 的材质化桌面外壳,提供顶栏、控制中心、通知中心、应用网格、媒体控件等,一并享有 Material You 的玻璃质感。社区里 NNN(Niri-Nix-Noctalia)与 Niri + DankMaterialShell 都是 2025 年最热门的开箱即用方案。 niri-wm 

# 安装 quickshell
yay -S quickshell-git

# 克隆 DankMaterialShell
git clone https://github.com/AvengeMedia/DankMaterialShell.git \
    ~/.config/quickshell/DankMaterialShell

# 在 niri config 里替换 waybar
spawn-at-startup "qs" "-c" "DankMaterialShell"

启动后用顶栏右上角的设置图标即可在 GUI 里改主题色、透明度、模糊强度,比手写 CSS 直观得多。另一个值得尝试的是 Noctalia。

十三、优化与排错

看不到模糊效果? Niri 没有窗口级模糊,只有终端 / 状态栏自身用 Wayland 协议申请的层级模糊会生效。要"全屏窗口都模糊"请改用 Hyprland;Niri 用户更现实的做法是把 Waybar/Fuzzel 的背景设为带半透明的纯色,搭配深色虚化壁纸已经非常通透。
NVIDIA 用户: 建议加 kernel 参数 nvidia_drm.modeset=1 与环境变量 LIBVA_DRIVER_NAME=nvidia、GBM_BACKEND=nvidia-drm、__GLX_VENDOR_LIBRARY_NAME=nvidia。
JetBrains/Electron 应用闪烁: 启动 xwayland-satellite,并在启动器命令前加 env -u WAYLAND_DISPLAY 强制走 X11。
截图工作流:
grim -g "$(slurp)" - | swappy -f - —— 区域截图后立即标注。
