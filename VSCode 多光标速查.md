# VS Code 多光标速查

常用多光标与块选择快捷键（Windows）：

- **鼠标多光标**: `Alt + 点击` → 在任意位置添加光标
- **选中下一个匹配**: `Ctrl + D` → 逐个添加相同词的光标
- **选中所有匹配**: `Ctrl + Shift + L` 或 `Ctrl + F2` → 选中全部匹配并进入多光标编辑
- **在上/下加入光标**: `Ctrl + Alt + ↑` / `Ctrl + Alt + ↓` → 上/下扩展光标
- **列/矩形选择（块选择）**: `Shift + Alt + 拖拽` 或 `Shift + Alt + ↑/↓/←/→` → 块选择
- **通过查找选中所有**: `Ctrl + F` 输入关键词 → `Alt + Enter`（或 `Ctrl + Shift + L`）选中所有结果
- **退出多光标**: `Esc` 或常规撤销 `Ctrl + Z`

macOS 对应：

- `Alt` 对应 `Option`，`Ctrl` 对应 `⌘` 在部分命令上（常见：`⌘ + D` 等）
- 常用替换：`Ctrl + Alt + ↑/↓` → `Option + ⌘ + ↑/↓`

小技巧：

- 先用 `Ctrl + D` 多次选中逐个修改，效率高且可回退。
- 组合 `Ctrl + D` 与 `Ctrl + Alt + ↓` 可以在多行相似位置快速批量编辑。
- 练习块选择处理对齐插入、列注释与代码列替换非常有用。

示例：

1. 将多行末尾添加相同后缀：`Shift + Alt` 拖动选择列，在每个光标输入内容即可。
2. 将变量名批量改为后缀形式：选中一个变量后 `Ctrl + D` 多次，然后直接编辑。
